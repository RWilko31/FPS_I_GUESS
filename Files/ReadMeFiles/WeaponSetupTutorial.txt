Add a new weapon to player:
1. add model to the weapon container on player
2. set weapon layer to playercamlayer
3. add data to the weapon table csv file.

Create a weapon pickup interactable for weapon:
1. add weapon model to level
2. set layer and tag to interactable (make sure to do both)
3. add a collider and set it to isTrigger = true
4. add interactable data script to object (and spinnyboi for sick spins)
5. set name of object to weaponpickup_*weapon name here*
note: if using multiple pickups add a space before any numbers e.g weaponpickup_gun (1)
